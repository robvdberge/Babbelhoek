# Babbelhoek

A fully object-oriented php forum in Dutch.

## Description

A fully-functional Forum in PHP.
Users are able to register, create topics and reply to topics.

### Uses

* Bootstrap v4.3.1
* Custom Template engine
* OOP MySQL CRUD engine
* Login functionality

### Programmed using

* WAMP 3.1.7 64bit
* MS Visual Studio Code
